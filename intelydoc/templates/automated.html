Automated Classification

Automated classification allows user to classify documents using a trained module. This is more suitable if large amount of documents present. There are two option for the user. First option is to use the Pre-trained Classifier which allows the user use classifiers which are available in the system. If you can't find the classifier of choice, User can go for the second option Custom Classifier which allows the user to train a classification model and to use it for the classification purposes. Steps for both classification are described.

Classification Using pre-trained classifer

Step 1

Select a Classifier

First step is selecting suitable classifier from the pre-trained classifier list

Step 2

Upload Documents

upload documents to be classified. There are two option to select documents. Entire folder can be selected using 'Select Folder' Option Multiple files can be selected using 'Select Files' option


Step 3

Remove Duplicates

All the duplicates will be visible. Duplicates can be removed in one click

Step 4

Classify and Download

After removing all duplicates you can Start classsification. Once
