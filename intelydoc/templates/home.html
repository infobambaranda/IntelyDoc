What is Document Classification


Document classification, sometimes known as categorization, is a way of classifying, classing, or categorizing documents based on the content within them. In essence, when a document is “captured” or comes into the business, document classification can be used to assess the content of the document and then put a category or class against it making the document easier to store and access.

Here documents are divided into user-defined categories using machine learning techniques. This can be either manual or automated and it can be used to easily sort any document type.

Both types of document classification have their advantages and disadvantages. On the one hand, classifying documents manually gives humans greater control over the process of classification, and they can make decisions as to which categories to use. However, when handling large volumes of documents, the manual process would not be more efficient. In such cases, it is much faster, more cost-efficient, to carry out automatic document classification.

Getting started with IntelyDoc


  • For a successful use of the application a steady internet connection should be there.
  • User should first register and log into the system in order to access the main features provided by the IntelyDoc.
  • Once the user logged in successfully, user can select either Keyword-based classification or Automated classification according to the requirement.
  • If Keyword-based classification is selected user can proceed to classification with following steps.
    • Step 1 – Upload documents to be classified
    • Step 2 – Identify duplicates and remove if necessary
    • Step 3 – Define categories with relevant keywords.
    • Step 4 – Perform classification and download classified documents.
  • If Automated classification is selected user have a choice to use a predefined classifier or to train a custom classifier. User can proceed to classification with following steps.
    • Step 1 – Select a classifier or train a classifier by providing categories with sample documents
    • Step 2– Upload documents to be classified
    • Step 3 – Identify duplicates and remove if necessary
    • Step 4 – Perform classification and download classified documents.
